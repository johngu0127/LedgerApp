# 簡易記帳本（Python）

一個可離線使用的個人支出記帳系統，支援 **多帳戶/錢包**、**類別管理**、**每日 / 每週 / 每月統計**，  
並可將資料儲存為 **CSV**。內建 Dashboard：**本月支出圓餅圖（按類別）** 與 **帳戶餘額**。

---

## 特色與對照需求

- ✅ 新增交易（收入 / 支出、金額、日期、說明）
- ✅ 類別紀錄（食物、娛樂、交通…可自訂）
- ✅ 查詢統計（每日 / 每週 / 每月）總支出
- ✅ 資料儲存：**CSV**
- ✅ Bonus：多帳戶 / 錢包（現金、玉山、台新…），Dashboard 圖表與帳戶餘額

---

## 安裝

需要 **Python 3.9+**（Windows 可用 Microsoft Store 或 python.org）。

安裝一次相依套件：

```bash
py -m pip install matplotlib
```

### 執行計劃

1. **放置專案資料夾**  
   將整個專案資料夾放在你要使用的位置，例如：`D:\MoneyData\LedgerApp`。

2. **開啟終端機 / PowerShell**  
   在該資料夾路徑開啟「命令提示字元」或 PowerShell。

3. **執行程式**

   ```bash
   py simple_ledger_gui.py
   ```
   
    如果你的電腦是用 python 指令，也可以改成：
  
    ```
    python simple_ledger_gui.py
    ```

4. **執行後會跳出記帳程式視窗。**

## 使用說明

以下是常用功能的簡要操作方式。

#### 1. 新增交易

在主畫面的 **「新增交易」** 區塊依序輸入：

- 日期：`YYYY-MM-DD`（例如：`2025-11-11`）  
- 類型：選擇 **income（收入）** 或 **expense（支出）**  
- 金額、類別、帳戶、說明  

填寫完成後，按下 **「新增」** 按鈕即可把交易加入列表。

#### 2. 自訂類別 / 帳戶

- 在「類別」或「帳戶」欄位輸入新的名稱並加入交易  
- 之後這個名稱會自動出現在下拉選單中，方便下次選取

#### 3. 查詢統計

- 右上方選擇 **「每日 / 每週 / 每月」** 與基準日期  
- 按下 **「計算支出」**  
- 會顯示該期間各類別的支出總額，方便查看花費分布

#### 4. 資料儲存 / 載入

- 按 **「儲存」**：將目前所有交易寫入 `ledger.csv`  
- 按 **「載入」**：從既有的 `ledger.csv` 匯入資料  
- 按 **「另存」**：以其他檔名備份目前資料

#### 5. Dashboard 儀表板

1. 切換到 **「Dashboard」** 頁籤  
2. 在輸入框輸入查詢月份，例如：`2025-11`  
3. 按下 **「更新圖表」**，畫面會顯示：

   - 左側：該月份的 **支出圓餅圖**（依「類別」分組），可快速看出哪一類支出最多  
   - 右側：各帳戶的 **累積餘額** 與 **本月淨變動** 狀況
